# 每日双语发布清单

## 内容

- [ ] `content/zh/` 已新增当日中文母稿，日期与文件名一致。
- [ ] 中文页的主标题、lead、统计、总判断、重点信号、风险边界与来源完整。
- [ ] 事实优先使用官方或一手来源，每条主张可追溯。
- [ ] 没有把日志、trace、签名身份或融资公告扩大成 ALUX 已交付能力。

## 英文精修

- [ ] 目标语言为美式英语（`en-US`），目标读者为全球 AI Agent、基础设施、技术与商业读者。
- [ ] 已读翻译配置与术语表。
- [ ] `content/en/YYYYMMDD.body.html` 已完成“分析 → 初译 → 审校 → 润色”。
- [ ] 英文页无未处理中文，读起来像母语级科技情报出版物，专业、简洁、准确、权威，不逐字直译。
- [ ] 中英数字、版本号、融资金额、公司名、产品名、RISC 结论和 ALUX 边界一致。
- [ ] 中英章节数、信号数、来源数与外链集合一致。
- [ ] 翻译清单的中文哈希、英文哈希已刷新，当期状态为 `reviewed`。

## 生成与页面

- [ ] 运行 `scripts/sync-reports.ps1` 成功。
- [ ] 中英首页的最新日期、标题、摘要、归档数量和月份数正确。
- [ ] 中英首页“最近更新”相同，使用最新审核记录对应的中国标准时间。
- [ ] `/daily/latest/` 与 `/daily/en/latest/` 指向同一日期。
- [ ] 每个日期页的“中文 / EN”切换保持同一期。
- [ ] 上一期/下一期、返回首页和最新一期链接可用。
- [ ] 浏览器页签显示 ALUX 三角标，不再显示默认小地球。
- [ ] canonical、hreflang、archive JSON 和 sitemap 正确。

## 多设备验收

- [ ] 1920、1440、1024、768、620、430、390、320 px 全部无横向溢出。
- [ ] 月份、日期范围、品牌名和导航不孤立断行。
- [ ] 中文不出现一个汉字独占末行，英文长专名和 URL 不裁切。
- [ ] 英文热区矩阵的标签与强度徽章没有越过左栏或遮住右栏正文；宽屏标签列不低于 `172px`，`620px` 及以下为单列。
- [ ] 英文面板标题与右侧说明没有重叠；`920px` 及以下上下排列。
- [ ] 顶栏 Logo 外框与语言切换外框均为 `44px` 高且垂直对齐，语言按钮点击高度不低于 `44px`。
- [ ] 日报页顶栏分隔线、Logo、品牌文字与正文网格左右对齐；宽屏、平板、手机分别跟随正文内边距。
- [ ] 全部英文历史页通过八档视口与 `621/920/921px` 断点的真实文字边界检查。
- [ ] 手机端保留最新一期与语言切换，主要点击区不低于 44 px。
- [ ] 检查首页、最早期、中间期和最新期的中英截图。
